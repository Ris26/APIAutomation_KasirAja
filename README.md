# How run  API Automation using mocha 

Installation

- npm install
- npm install supertest
- npm install mocha
- npm install chai

How to run

- npx mocha api/test/{file name}.js

How to generate report

- npx mocha api/test/products/crud_Products.js --timeout=30000 --reporter mochawesome
